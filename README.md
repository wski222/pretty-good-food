# Pretty Good Food — Interactive Prototype

Open `index.html` in Chrome, Safari, or Firefox.

This version includes:
- Modernized Pretty Good Food branding
- Compact home introduction and feed-first layout
- House Rules timeline and two-column full scoring guide
- Food and cocktail feeds, filters, detail pages, and social metadata
- More visible rating badges
- Simplified Add flow with rating and review together
- Paste-friendly Ingredients and Directions text areas
- Multiple photo upload previews
- Ingredient-page and directions-page photo uploads for cookbook recipes
- Made It Again cooking history

This is still a front-end prototype. Data is sample content and is not connected to accounts or cloud storage.
